## [1.0.3] - 2026-06-24

> **功能更新**：新增「笔记 / 知识库」与「专注 / 番茄钟」两大模块，并为协作聊天加上表情回应。建议升级体验新功能。

### 新增

- **笔记 / 知识库（Obsidian 式）**：本地优先的真 `.md` 仓库（可直接用 Obsidian 打开）；双链 `[[..]]` 与反链、Live Preview 编辑（CodeMirror 6）、全文检索、标签嵌套树、可编辑属性（frontmatter）。
- **关系图谱**：全局 / 局部视图，悬停高亮邻居，**点击节点平滑居中放大**；并支持**全库 AI 自动连线**（让 AI 分析全库、建议并批量建立双链）。
- **AI 辅助写作**：接入 Codex（OpenAI Responses API，流式）——扩写 / 续写 / 总结 / 润色 / 改写 / 起标题 / 翻译 / 标签建议 / RAG 问答；选区或 ⌘K 内联改写，带 diff 评审（保留 / 放弃 / 重试 / 撤回）。
- **Canvas 白板 + Bases 数据库视图**：JSON Canvas 白板、笔记的表格 / 看板式视图；命令面板、快速跳转、日记。
- **笔记云端同步**：整库 blob + 三方合并的单会话顺序模型（写入云端、另一端打开时与本地对比合并）。
- **专注 / 番茄钟**：计时器（专注 / 短休 / 长休）、任务绑定、白噪音（白 / 棕 / 雨声）、专注统计与连续天数；**团队专注排名**（今日 / 本周）。
- **协作消息表情回应**：Telegram 式全 emoji 回应，长按 / 悬停拾取，回应聚合显示。
- **可隐藏内容入口** + **左侧导航长按拖动排序**（顺序跨设备记忆）。

### 修复

- 各群协作聊天严格隔离，不再串台；Electron 下用应用内输入弹窗替换不可用的 `window.prompt`。
- AI 接口遇上游过载 / 限流 / 超时自动退避重试并友好提示；Canvas 夜间配色修正。

### 备注

- 笔记云同步、团队专注排名、消息表情回应依赖协作服务端对应能力（已部署的服务器自动生效，否则相应功能自动降级 / 仅本地）。
- macOS 为「提示下载安装包」（dmg）方式，无原地自动更新。

## [1.0.2] - 2026-06-23

> **可选更新**：新增日历 / 待办 等效率模块；旧版本仍可正常使用，非必需升级。

### 新增

- **个人日历**：月 / 周 / 日视图，事件增删改、多日历（颜色 / 显隐）、当前时间线、重复事件；支持**导入外部日历（.ics）**。
- **组队日历**：团队共享事件、邀请成员与 接受 / 拒绝 / 待定（RSVP）、成员筛选；经协作服务器同步，未登录可本地预览。
- **备忘录 / 待办**：清单 + 智能视图（今天 / 最近 7 天 / 收件箱 / 全部 / 已完成）、优先级 / 标签 / 子任务 / 重复；**自然语言快速添加**（中文连写如「明天下午5点写周报 !high #工作」，基于 chrono-node）；便签看板（置顶 / 配色）。
- **跨功能打通**：待办可「加入日历 / 一键同步到日历」，个人事件可「共享到团队」；新手引导覆盖新模块。
- **个人日历 / 待办云端同步**：按用户多端同步、WebSocket 实时、版本号乐观并发**防止旧版本覆盖新版本**（服务端 `/api/user-store`；需部署服务端后生效，未支持时自动「仅本地」）。

### 修复

- 登录后偶发**「账号在别处登录」**：云同步/组队日历不再自建第二条 WebSocket，全应用复用聊天的唯一连接。
- **协作聊天历史 / 成员按「群」隔离**：同一客户端连不同协作服务器不再串台，登录即切换。
- 快速添加**中文连写「分词」**：改用 chrono-node，连写也能正确识别日期/时间并剥离出干净标题。

### 优化

- 侧栏导航过多时可滚动（隐藏滚动条），收起按钮与版本卡固定底部；三大模块整体字体放大、便签 UI 重做（参考 Keep，任意底色文字可读）。

## [1.0.1] - 2026-06-22

> **可选更新**：本版以体验优化与小问题修复为主，旧版本仍可正常使用，非必需升级。

### 新增

- **未登录预览**：登录页新增「先不登录，随便逛逛」，可进入只读主界面预览；顶部预览条可随时返回登录。
- **新手引导**：首次进入主界面自动弹出分步高亮导览，逐一介绍核心面板；标题栏「?」可随时重看。
- **内置网页右键菜单**：内嵌 AI 网页支持浏览器式右键菜单（后退 / 前进 / 重新加载、剪切 / 复制 / 粘贴、复制链接、在浏览器打开、复制图片、检查元素等）。
- **登录页改版**：加入品牌欢迎区与功能简介，开局更友好。

### 修复

- **代理重启端口冲突**：修复重启 sing-box 时偶发的端口占用（`bind: only one usage...` / EADDRINUSE）导致代理起不来、反复重启失败；现等旧进程退出、端口释放后再启动，并避免旧进程退出时误清新进程引用。
- **「一键加入并重启」不生效**：修复默认（all）模式下自动加入的域名（`auto_domains`）未并入实际代理路由、且代理检测始终判为「回落」的问题；现路由规则与代理检测共用同一份域名清单，加入并重启后即可正确走代理并翻绿。

### 备注

- 本版不强制更新。AI 网页内的翻译工具与共享协作空间已在规划中（见 `docs/`）。

## [1.0.0] - 2026-06-19

首个正式版本。

### 新增

- 整合 **ChatGPT / Claude / Gemini** 三家 AI 网页于一个客户端（多标签、各自独立会话、登录态持久化，入口可在设置中开关）。
- **统一代理**：基于 sing-box，按内置域名清单仅对 AI 站点走代理；管理员统一下发连接配置，成员首登自动拉取，无需手配。
- **代理检测**：实时显示页面流量是否全部走代理；发现「会用到却没走代理」的域名时自动加入本机清单并上报管理员，重启即时生效。
- **可选机场订阅模式**：管理员粘贴 Clash 订阅、选节点下发，客户端可选择走机场节点（与统一代理并存，默认统一）。
- **团队协作聊天**：私聊 / 房间消息、图片与文件、撤回 / 已读 / 回复 / 转发、离线补同步、可自定义提醒。
- **使用统计**：按 ChatGPT / Gemini / Claude 维度统计每人查询量与排行。
- **管理控制台**：用户管理、Sender 默认配置下发、机场节点下发、用户反馈查看、漏走代理域名汇总、版本发布。

### 变更

- **自动更新改为以 GitHub Releases 为更新源**（参考 [cc-switch](https://github.com/farion1231/cc-switch)），不再经过任何自建服务器。
- **Windows 原地无感更新**：后台下载、自动安装并重启，快捷方式与安装位置不变，账号 / 聊天记录 / 网页登录态全部保留；macOS 暂为下载安装包方式。

### 备注

- 更早的 5.x 为测试版本，不在此正式记录。
